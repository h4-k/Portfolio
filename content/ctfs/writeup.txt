5. EXPLOITATION STRATEGY
------------------------
Approach: DNS Poisoning + Controlled Web Server

Step 1: DNS Poisoning
- Modified /etc/hosts to point google.com to attacker-controlled IP
- Command: echo "192.168.128.53 google.com" > /etc/hosts

Step 2: Web Server Setup
- Set up Python HTTP server on attacker machine
- Served malicious HTML with payload in title tag

Step 3: Payload Delivery
- Cron job fetches from google.com (now attacker-controlled)
- Extracts title and writes to /script2.sh
- Payload: "/bin/bash"

Step 4: Root Shell Execution
- Executed: sudo /script2.sh
- Gained root shell due to sudo permissions
